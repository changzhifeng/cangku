输入cls 按回车，即可清除当前屏幕。 输入npm install xxx 按回车即可安装xxx相应的模块。 注意：安装但不写入package.json中。 输入npm install -g xxx 按回车即可安装xxx模块到全局。 输入npm install --save-dev xxx || npm i -D xxx 按回车即可将xxx模块安装到开发环境下。 注意：安装并写入package.json的"devDependencies"中。 输入npm install --save xxx || npm i -S xxx 按回车即可将xxx模块安装到生产环境下。 注意：安装并写入package.json的"dependencies"中。 输入npm uninstall xxx 按回车即可删除xxx模块。 输入npm uninstall -g xxx 按回车即可删除全局模块xxx。 输入npm install -g cnpm --registry=https://registry.npm.taobao.org 按回车即可安装npm镜像，以后再用到npm的地方直接用cnpm来代替就好了。 输入vue -V || vue --version 按回车，查询vue脚手架的版本号。 输入node -v || node --version 按回车，可以看见当前node的版本。 输入npm -v || npm --version 按回车，可以看见当前npm的版本。 如果你的系统安装了多个版本的node，可以输入 输入where node 按回车，可以查看当前命令行使用node命令用的是哪个文件夹里面的node。
